# Категории и анализ времени

## Создание категории

Категория - это способ группировки временных интервалов, у каждого интервала только одна категория. 

![Экран для работы с категориями](./categories.jpg)

Чтобы создать категория:

- [ ] **Наберите** название новой категории
- [ ] **Нажмите** кнопку для добавления
- [ ] **Введите** текст для поиска по категориям
- [ ] **Нажмите** иконку корзины чтобы удалить категорию

## Работа с анализом

- [ ] **Откройте** страницу анализа
- [ ] **выберите** тип анализа
- [ ] **Введите** категории
- [ ] **Установите** период
