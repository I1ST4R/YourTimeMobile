# Начало работы

Эта страница поможет вам сделать первые шаги в **Your Time**. 

## Установка и настройка

1.  Скачайте apk из [GitHub](https://github.com/I1ST4R/YourTime) 
2.  Уже можно использовать, но для полного экспириенса предлагаем зарегистрироваться

## Ваш первый интервал времени

- [ ] **Нажмите** кнопку "+" на главном экране
- [ ] **Поменяйте** название интервала и его категорию
- [ ] **Укажите** время или воспользуйтесь таймером
- [ ] **Установите** дату
