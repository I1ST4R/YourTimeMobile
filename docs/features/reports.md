# Аналитика и отчеты

Этот раздел поможет вам *разобраться* в системе отчетности **Your Time** и научиться извлекать максимум пользы из собранной статистики.

## Типы отчетов

### Ежедневная статистика
- *Общее время работы*
- *Распределение* по категориям в течении дня

### Аналитика с указанием периода (от 2 дней и больше)
- **Сравнение** продуктивности по дням
- *Изменение* по одной категории

## Настройка отчетов

Перед генерацией отчета убедитесь что:

- [ ] **Выбраны** нужные категории для анализа
- [ ] **Указан** корректный период времени
